## AI 프롬프트 활용 3-20: 마이크로파이썬에서 list의 항목을 모두 출력하는 프로그램
## 프롬프트 
```
마이크로파이썬에서 list 자료형으로 임의의 요소를 5개 만들고, 모두 출력하는 코드를 작성해 줘.
```
## AI가 만든 결과
아래는 마이크로파이썬(MicroPython)에서 리스트 자료형을 생성하고, 임의의 요소 5개를 담은 뒤 모두 출력하는 간단한 예시 코드입니다.

```python
# 리스트 생성 및 임의의 요소 5개 추가
my_list = ["Hello", 42, 3.14, True, [1, 2, 3]]

# 리스트에 담긴 요소 전체 출력
for element in my_list:
    print(element)
```

### 설명
1. `my_list`라는 리스트를 만들고, 다음과 같은 임의의 자료형 5개를 넣었습니다.
   - 문자열: `"Hello"`
   - 정수: `42`
   - 부동소수점: `3.14`
   - 불리언: `True`
   - 리스트: `[1, 2, 3]`

2. `for element in my_list:` 구문을 사용해 리스트에 담긴 각각의 요소(`element`)를 순회하며 `print()` 함수로 출력합니다.

이 코드를 MicroPython REPL 혹은 스크립트 파일로 실행하면, 각 요소가 줄바꿈되어 순서대로 출력됩니다.
